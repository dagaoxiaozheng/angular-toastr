# angular-toastr
提示组件 : angular-toastr 

+ 引入文件 （angular.min.js和angular-animate.min.js必须是同一版本） [下载地址](https://code.angularjs.org/)

```
<link rel="stylesheet" href="./angular-toastr.min.css">
<script src="./angular.min.js"></script>
<script src="./angular-animate.min.js"></script>
<script src="./angular-toastr.tpls.min.js"></script>
```

+ 依赖模块

```
var app = angular.module('app', ['ngAnimate', 'toastr'])
```
